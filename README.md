Содержит 5 арифметических игр: 

Калькулятор

Арифметическая прогрессия

Для установки игра необходимо выполнить команду:

uv tool install dist/*.whl

После установки игры станут доступны по следующим командам: 

brain-calc
brain-even
brain-progression
brain-gcd
brain-prime
